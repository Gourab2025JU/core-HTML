List of HTML Tags

  1. <html> - The root element of an HTML document. It wraps all the content on the web page.
  2. <head> - Contains meta-information about the HTML document, such as the title and links to stylesheets.
  3. <title> - Sets the title of the HTML document, which appears in the browser tab.
  4. <meta> - Provides metadata about the HTML document, such as character encoding or viewport settings.
  5. <body> - Contains the content of the HTML document, such as text, images, and links.
  6. <h1> - Defines the most important heading. Headings range from <h1> to <h6> with decreasing importance.
  7. <p> - Defines a paragraph of text.
  8. <a> - Creates a hyperlink to another webpage or resource. It uses the href attribute to specify the URL.
  9. <img> - Embeds an image in the document. It uses the src attribute to specify the image file and alt for alternative text.
  10. <ul> - Defines an unordered list. Items in the list are typically marked with bullets.
  11. <ol> - Defines an ordered list. Items are numbered.
  12. <li> - Defines a list item. It is used within both ordered and unordered lists.
  13. <div> - Defines a division or section in the document. It is often used as a container for styling or scripting purposes.
  14. <span> - Defines a small section of text or other content. It is typically used for styling a part of the text.
  15. <br> - Inserts a line break in the text.
  16. <hr> - Creates a horizontal rule or line to separate content.
  17. <form> - Defines an HTML form for user input. It can include various form elements like text fields, buttons, and checkboxes.
  18. <input> - Defines an input field where users can enter data. It can take various types such as text, password, or checkbox.
  19. <button> - Defines a clickable button. It can be used for submitting forms or triggering JavaScript functions.
  20. <table> - Defines a table. It is used to display tabular data.
  21. <tr> - Defines a row in a table.
  22. <td> - Defines a cell in a table row.
  23. <th> - Defines a header cell in a table, which is typically bold and centered.
  24. <style> - Contains CSS styles to apply to the HTML document.
  25. <script> - Contains JavaScript code or links to an external JavaScript file.
